# Ready To Plot

This folder contains the app-ready visualization outputs for the Sacramento-Davis-Fairfield demo.

Files:
- `map_fire_points.jsonl`: FIRMS points for map layers/heatmaps
- `map_farm_status.jsonl`: latest farm marker status (risk score + level)
- `map_farm_boundaries.geojson`: farm boundaries as GeoJSON features (or inferred grid cells)
- `chart_farm_timeseries.jsonl`: hourly farm risk/features for charts
- `manifest.json`: file paths + record counts from the build step
